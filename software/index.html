GOexpress Bioc-release: Bioc-devel:

Methods to visualise the expression profile of genes from transcriptomics experiments, and offers a supervised clustering approach to identify GO terms containing genes with expression levels that best classify two or more predefined groups of samples.
iSEE Bioc-release: Bioc-devel:

Functions for creating an interactive graphical user interface based on RStudio Shiny for exploring data stored in SummarizedExperiment objects. Particular attention is given to visualization of dimensionality reduction results in SingleCellExperiment objects.
