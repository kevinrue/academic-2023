Software

R packages

I maintain or co-develop the following R packages.


 BiocCheck executes Bioconductor-specific package checks.


deeperTools provides convenience functions to process files produced by deepTools.

GOexpress provides methods to visualise the expression profile of genes from transcriptomics experiments, and offers a supervised clustering approach to identify GO terms containing genes with expression levels that best classify two or more predefined groups of samples.


hancock aims to provide a collection of methods for learning and applying gene signatures associated with cellular phenotypes and identities.

iSEE provides functions for creating an interactive graphical user interface based on RStudio Shiny for exploring data stored in SummarizedExperiment objects, including row- and column-level metadata. Particular attention is given to single-cell data in a SingleCellExperiment object with visualization of dimensionality reduction results.


sqlighter provides convenient wrapper functions around the RSQLite package.

TVTB provides S4 classes and methods to filter, summarise and visualise genetic variation data stored in VCF files.


unisets provides a collection of S4 classes to store relationships between elements and sets, with a particular emphasis on gene sets.


xCellData uses the Sets class defined in the unisets package (see above) to represent 489 cell type gene signatures from Aran et al., 2017.
