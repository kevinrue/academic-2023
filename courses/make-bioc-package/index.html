Syllabus

Overview

This course demonstrates the process of creating a Bioconductor package, from essential requirements to advanced DevOps practices that facilitate maintenance and dissemination.

This course is presented as sequence of steps that establish the essential components of a minimal Bioconductor package, accompanied by a additional sections to extend the package set-up with additional valuable functionality.

Pre-requisites:

  1. Install R
  2. Install RStudio
  3. Install version control software

Essentials:

  1. Initialize an R package skeleton
  2. Initialize version control
  3. Write your first function
  4. Write help pages for your functions
  5. Test your functions
  6. Write a vignette

Advanced:

  1. Enable continuous integration
  2. Track code coverage
  3. Deploy documentation online
  4. Adding badges to your README
  5. Making a Docker container

Readers are encouraged to contribute feedback through issues.

Pre-requisites

  • Basic knowledge of R syntax.
  • Basic knowledge of git.
  • Familiarity with GitHub.

Additional background reading about the programming environment, relevant packages, and example use cases:

  • R packages, by Hadley Wickham and Jennifer Bryan: https://r-pkgs.org/index.html

R / Bioconductor packages used

  1. devtools : https://CRAN.R-project.org/package=devtools

Course goals and objectives

Learning goals

  • Outline the unique features of Bioconductor and CRAN package.
  • Identify best practice in Bioconductor package development.
  • Utilize R packages designed to facilitate the rapid and consistent development of R packages.
  • Describe the components and functionality of R and Bioconductor packages.

Learning objectives

  • Memorize the key components and requirements of a Bioconductor package.
  • Set up a working continuous integration environment for durable development and maintenance.
  • Interact with tools that facilitate the development, maintenance, and deployment of a Bioconductor package.
  • Practice the workflow to write a Bioconductor package from scratch.
